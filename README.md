# 🎲 Two Player Dice Game

A fun and interactive **Two Player Dice Game Web Application** built using HTML, CSS, and JavaScript. Two players can roll the dice, compare their scores, and find out the winner. The game uses random dice generation along with dice images to provide an engaging visual experience.

## ✨ Features

- 🎲 Two-player dice rolling gameplay
- 🧑‍🤝‍🧑 Player 1 vs Player 2 competition
- 🖼️ Realistic dice images for visual feedback
- 🎯 Random dice number generation using JavaScript
- 🏆 Automatic winner announcement
- 🔄 Restart and replay functionality
- 📱 Responsive design for different devices
- ⚡ Simple and interactive user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Dice images and other visual assets

## 🎮 How to Play

1. Open the game in your web browser.
2. Click the **Roll Dice** button.
3. Both Player 1 and Player 2 receive a random dice value.
4. The player with the higher dice number wins.
5. If both players get the same number, the result is a draw.
6. Restart the game to play another round.
